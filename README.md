# Node, Express Server, and ES6 using babel.

# Features
 * babel 7
 * Express
 * bcrypt
 * jsonwebtoken
 * mongoose
 
# Requirements
 * node v13
 * mongodb compass
 
# Installation
 * `git clone https://github.com/kantivekariya/express-mongoose-es6-rest-api.git`
 * `cd express-mongoose-es6-rest-api`
 * `npm install`
 * `npm run dev` Rus as Development
